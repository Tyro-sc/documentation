Item

Item is used for the elements listed in Dropdown, ListBox and ListView

Item in ListBox or Dropdown

States Properties Actions

selected

value

[select]

unselected

[unselect]

Usage:

cities.items()[0].should {
    have value('Montpellier')
    be selected
}

cities.item('New York').should {
    be disabled
    be unselected
}
